This framework deals with the forensic timeline generation for Android platform.

The visualisation of timeline is powered by d3.js and user interactions are carried by jQuery, Zeptop and OpenTips.
The web server is backed by node.js, details of packages can be seen from package.json in the server folder.
An Android App and several scripts (shell, python) are involved for data extraction and parsing.
